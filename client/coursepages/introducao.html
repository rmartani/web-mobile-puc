WEB para Dispositivos Móveis

Sobre

Nesse curso iremos abordar os conceitos básicos para se começar a desenvolver para Web levando em consideração seus padrões e arquitetura, bem como nas técnicas disponíveis para renderização de seu site ou sistema Web em diferentes dispositivos, seja pelo pelo modo responsivo, ou adaptativo.

Agenda

  • JQuery
  • Padrões Web
  • XDK
  • Responsividade
  • CSS para desenvolvimento de páginas Web em dispositivos móveis
  • Fundamentos de arquitetura de sistemas web
  • Desenvolvimento de páginas Web usando HTML5 para dispositivos móveis: listas, tabelas, links, formulários e imagens

  2. Trabalho Prático - Entrega Github - 30 pontos - Dia 29/05/2017

    Como um projeto agile, com escopo aberto, a cada semana de acordo com as matérias estudadas novas estórias serão adicionadas. Até no final das nossas aulas teremos um trabalho prático sobre os assuntos da ementa e das informações compartilhadas entre a turma.

    Pode-se utilizar a API Randomica de Usuário randomuser.me ou nesse projeto do Material Ditático tem um sample de uma API em Node.JS

    O trabalho deve ser feito em grupo, favor enviar um email com as informações do grupo e dos participantes até o final da primeira semana do curso.

    O código deve ser hospedado desde o início no Github, e feito em coloboração. Favor levar em consideração os diferentes níveis de conhecimento da matéria entre os alunos na hora de criar os grupos. O objetivo é ter pessoas com níveis diferentes de conhecimento no grupo.

    Favor enviar juntamente com o nome do grupo o endereço do repositório.

    Qualidade do código será avaliado.
